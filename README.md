# devops-automation-toolkit

Practical Bash and Python automation tools for Cloud and DevOps workflows.

## What this repo covers

- Environment validation scripts
- System health checks
- Log parsing and analysis
- JSON and API utilities
- Terraform wrapper scripts
- Docker helper scripts
- CI/CD automation
- Basic Ansible examples

## Repo structure

```
bash/          - Shell scripts
python/        - Python automation tools
docs/          - Notes and references
sample_data/   - Test inputs and mock data
tests/         - Script and tool tests
```

## Status

In progress. Scripts are added incrementally as working, tested tools.
